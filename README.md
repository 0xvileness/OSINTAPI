OSINT API is an based on Intel leaks api.




Functions: 

Query by email, username, domain, affected URLs or IP address

Displays detailed information of infected equipment:

Date of commitment

Computer name and operating system

Passwords and logins found

Malware detected

Intuitive interface with interactive menu

Formatted results for better readability




Installation: 

git clone https://github.com/0xvileness/OSINTAPI.git

cd OSINTAPI

pip3 install -r requirements.txt
